# 图片说明

请在此文件夹下放置以下图片：

## 公司LOGO
放置在 `images/` 文件夹下，命名为：

- **logo.png** - 公司LOGO
  - 建议尺寸：300 x 120 像素（或按比例）
  - 格式：PNG（支持透明背景）或 JPG
  - 用于导航栏和首页顶部展示

2. **banner2.jpg** - 第二张轮播图

   - 建议尺寸：1920 x 500 像素
   - 建议主题：携手共创辉煌

## 设备展示图片（3张）

## 图片要求

- 格式：JPG 或 PNG
- 文件大小：建议每张图片不超过 500KB，以确保加载速度
- 命名：严格按照上述名称命名，区分大小写
- 位置：所有图片都放在 `web2/images/` 文件夹下

## 使用方法

1. 准备好符合要求的图片
2. 将图片重命名为对应的文件名（如 `banner1.jpg`、`device1.jpg` 等）
3. 将图片放入 `web2/images/` 文件夹
4. 打开网页即可看到图片效果
